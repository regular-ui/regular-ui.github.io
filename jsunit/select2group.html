API

Class

Select2Group

继承自Component。

Options

new Select2Group()
参数类型默认值描述
dataobject绑定属性
data.sourceobject[][]数据源
data.source[].namestring每项的内容
data.selectedobjectnull最后的选择项
data.selectedItemsobject[][]所有的选择项
data.placeholdersstring[][]默认项的文字
data.readonlybooleanfalse是否只读
data.disabledbooleanfalse是否禁用
data.visiblebooleantrue是否显示
data.classstring''补充class
serviceobject数据服务

Events

select2Group.$on('change')

最后的选择项改变时触发

属性类型描述
selectedobject最后的选择项
selectedItemsobject所有的选择项
select2Group.$on('select')

选择某一项时触发

属性类型描述
selectedobject当前选择项
levelobject当前选择的层级
