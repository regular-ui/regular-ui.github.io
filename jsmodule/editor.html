编辑器 Editor

示例

基本形式

<editor />

API

Class

Editor

继承自Component。

Options

new Editor()
参数类型默认值数据流向描述
dataobject绑定属性 | Binding Properties

Methods

editor.close(result)

关闭模态对话框

参数类型默认值描述
resultboolean点击确定还是取消
返回值类型描述
无返回值

Events

editor.$on('close')

关闭对话框时触发

属性类型描述
resultboolean点击了确定还是取消
editor.$on('ok')

确定对话框时触发

属性类型描述
无属性
editor.$on('close')

取消对话框时触发

属性类型描述
无属性
