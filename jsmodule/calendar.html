日历 Calendar

示例

基本形式

<calendar />
<calendar date="2008-08-08" />

禁用组件

<calendar disabled={true} />

日期范围

<calendar minDate={minDate} maxDate={maxDate} />
<calendar minDate="2008-08-08" maxDate="2008-08-16" />
<calendar ref="customCalendar" />

var component = new RGUI.Component({
    template: template,
    data: {
        minDate: new Date(+new Date + 2*24*3600*1000),
        maxDate: new Date(+new Date + 7*24*3600*1000)
    },
    init: function() {
        // 只能选择星期一、三、五
        this.$refs.customCalendar.isOutOfRange = function(date) {
            return !(date.getDay()%2);
        }
        this.$refs.customCalendar.$update();
    }
});

API

Class

Calendar

继承自Component。

Options

new Calendar()
参数类型默认值描述
dataobject绑定属性
data.dateDate | stringTODAY当前选择的日期
data.minDateDate | stringnull最小日期，如果为空则不限制
data.maxDateDate | stringnull最大日期，如果为空则不限制
data.readonlybooleanfalse是否只读
data.disabledbooleanfalse是否禁用
data.visiblebooleantrue是否显示
data.classstring''补充class
