输入扩展 Input2

示例

基本形式

<input2 rules={rules} />

var component = new RGUI.Component({
    template: template,
    data: {
        rules: [
            {type: 'isRequired', message: '请输入字符！'},
            {type: 'isLength', min: 5, max: 8, message: '请输入5-8个字符！'}
        ]
    }
});

API

Class

Input2

继承自Component。

Options

new Input2()
参数类型默认值描述
dataobject绑定属性
data.valuestring''输入框的值
data.typestring''输入框的类型
data.placeholderstring''占位符
data.rulesobject[][]验证规则
data.readonlybooleanfalse是否只读
data.disabledbooleanfalse是否禁用
data.visiblebooleantrue是否显示
data.classstring''补充class
