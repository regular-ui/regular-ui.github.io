日期时间选择 DateTimePicker

示例

基本形式

<dateTimePicker />
<dateTimePicker date="2012-12-21 12:21" />

禁用组件

<dateTimePicker disabled={true} />

日期范围

<dateTimePicker minDate={minDate} maxDate={maxDate} />
<dateTimePicker minDate="2008-08-08 12:00" maxDate="2008-08-12 14:45" />

var component = new RGUI.Component({
    template: template,
    data: {
        minDate: new Date(+new Date + 2*24*3600*1000),
        maxDate: new Date(+new Date + 7*24*3600*1000)
    }
});

数据绑定

<dateTimePicker date={date} />
<dateTimePicker date={date} />
<p>当前选择的日期为：{date | format: 'yyyy-MM-dd HH:mm'}</p>

事件

请打开浏览器的控制台查看结果。

<dateTimePicker
    on-toggle={console.log('on-toggle:', '$event.open:', $event.open)}
    on-change={console.log('on-change:', '$event.date:', $event.date)} />

API

Class

DateTimePicker

继承自Dropdown。

Options

new DateTimePicker()
参数类型默认值描述
dataobject绑定属性
data.dateobjectnull当前选择的日期时间
data.placeholderstring'请输入'文本框的占位文字
data.minDateDate | stringnull最小日期时间，如果为空则不限制
data.maxDateDate | stringnull最大日期时间，如果为空则不限制
data.readonlybooleanfalse是否只读
data.disabledbooleanfalse是否禁用
data.visiblebooleantrue是否显示
data.classstring''补充class

Methods

dateTimePicker.isOutOfRange(date)

是否超出规定的日期时间范围

参数类型默认值描述
dateDate待测的日期时间

Inherited Methods

@override dateTimePicker.select(item)
dateTimePicker.toggle(open)

在展开/收起状态之间切换

参数类型默认值描述
openboolean展开/收起
dateTimePicker.$updateSource()

从service中更新数据源

参数类型默认值描述
无参数

Events

dateTimePicker.$on('change')

日期时间改变时触发

属性类型描述
dateobject改变后的日期时间

Inherited Events

@override dateTimePicker.$on('select')
dateTimePicker.$on('toggle')

展开/收起时触发

属性类型描述
openobject展开/收起状态
