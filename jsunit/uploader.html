上传 Uploader

示例

基本形式

<uploader url="/api/upload" />

修改标题

<uploader title="上传文件" url="/api/upload" />

按钮自定义

<uploader url="/api/upload">
    <a class="u-btn u-btn-primary">上传 <i class="u-icon u-icon-upload"></i></a>
</uploader>

禁用组件

<uploader url="/api/upload" disabled />

文件类型限制

<uploader url="/api/upload" extensions="jpg,gif,png" />

API

Class

Uploader

继承自Component。

Options

new Uploader()
参数类型默认值数据流向描述
dataobject绑定属性
data.titlestring''outer => inner按钮文字
data.urlstring''outer => inner上传路径
data.dataTypestring'json'outer => inner数据类型
data.dataobjectouter => inner附加数据
data.namestring'file'outer => inner上传文件的name
data.extensionsstring
string[]		''outer => inner可上传的扩展名，如果为空，则表示可上传任何文件类型
data.disabledbooleanfalseouter => inner是否禁用
data.visiblebooleantrueouter => inner是否显示
data.classstring''outer => inner补充class

Methods

uploader.upload()

弹出文件对话框并且上传文件

参数类型默认值描述
无参数
返回值类型描述
无返回值

Events

uploader.$on('sending')

发送前触发

属性类型描述
sourceobject事件发起对象
dataobject待发送的数据
uploader.$on('error')

上传错误时触发

属性类型描述
sourceobject事件发起对象
errorobject错误
uploader.$on('success')

上传成功时触发

属性类型描述
sourceobject事件发起对象
dataobject返回的数据
uploader.$on('complete')

上传完成时触发

属性类型描述
sourceobject事件发起对象
xmlobject返回的xml
