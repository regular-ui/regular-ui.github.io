时间选择 TimePicker

示例

基本形式

<timePicker />
<timePicker time="15:45" />

禁用组件

<timePicker disabled={true} />

日期范围

<timePicker minTime="12:00" maxTime="14:45" />

数据绑定

<timePicker time={time} />
<timePicker time={time} minTime="18:00" maxTime="19:30" />
<p>当前选择的时间为：{time}</p>

事件

请打开浏览器的控制台查看结果。

<timePicker on-change={console.log('on-change:', '$event.time:', $event.time)} />

API

Class

TimePicker

继承自Component。

Options

new TimePicker()
参数类型默认值描述
dataobject绑定属性
data.timestring'00:00'当前的时间值
data.minTimestring'00:00'最小时间
data.maxTimestring'23:59'最大时间
data.readonlybooleanfalse是否只读
data.disabledbooleanfalse是否禁用
data.visiblebooleantrue是否显示
data.classstring''补充class

Methods

timePicker.isOutOfRange(time)

是否超出规定的时间范围

参数类型默认值描述
timeTime待测的时间

Events

timePicker.$on('change')

时间改变时触发

属性类型描述
timeobject改变后的时间
